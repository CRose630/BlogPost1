How to: Scaffold a Project

Using the command prompt terminal, connect to the LAMP server by inputting the following command into the command terminal: $ ssh username@64.227.19.180

If this is your first time accessing the LAMP server, you may be asked if you are sure that you want to connect; if this is the case, type yes.
Then input your password. From here, access the web root with the following command: $ cd ../../var/www/html

Then, create a directory named the same as your username, and change it to your current working directory.
From here, create your first project using the mkdir command. Change it to your current working directory, and make subfolders and files as you see fit. Using the touch command, create an index.html file. Use the nano command to edit it.
